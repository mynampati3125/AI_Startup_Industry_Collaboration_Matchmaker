# AI Startup–Industry Collaboration Matchmaker

## Overview
This project was developed during a 24-hour hackathon at PBR VITS, Kavali.

The system intelligently matches startups with industries using semantic similarity and risk-based ranking mechanisms.

## Problem Statement
Startups often struggle to identify the right industry partners due to:
- Technical misalignment
- Budget incompatibility
- Risk uncertainty

## Solution
An AI-powered matchmaking engine that:
- Uses transformer-based sentence embeddings
- Applies cosine similarity for compatibility scoring
- Incorporates risk-based ranking logic
- Filters based on budget and technology requirements

## Tech Stack
- Python
- Sentence Transformers
- Cosine Similarity
- Streamlit
- Pandas

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py
